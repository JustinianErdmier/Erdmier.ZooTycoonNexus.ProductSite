# Home Page Implementation Plan

> - **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development`.
> - Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the Zoo Tycoon Nexus marketing landing page as a Blazor WASM standalone application using proper component composition and the project's naming and type
conventions.

**Architecture:** `Home.razor` composes ten section components inside a `<div class="nexus">` wrapper. All CSS is already present in `wwwroot/css/app.css`. Smooth-scroll anchor
navigation is handled via a tiny JS interop helper (`wwwroot/js/site.js`). The email signup form is purely in-memory state with no persistence.

**Tech Stack:** .NET 10 Blazor WASM, `IJSRuntime` for JS interop, C# positional records for data models, `MarkupString` for developer-authored inline HTML in roadmap content.

**Spec:** `Docs/Superpowers/Specs/2026-06-07-home-page-design.md`

```csharp
namespace Erdmier.ZooTycoonNexus.Presentation.Web.Common.Extensions;

/// <summary>Extension methods for <see cref="IniEntryKinds"/>.</summary>
public static class IniEntryKindsExtensions
{
    /// <summary>Maps an <see cref="IniEntryKinds"/> value to its corresponding CSS class token for use in a feature card's INI snippet.</summary>
    /// <param name="kind">The INI entry kind to map.</param>
    /// <returns>
    /// <c>"k"</c> for <see cref="IniEntryKinds.Key"/>;
    /// <c>"ok"</c> for <see cref="IniEntryKinds.Ok"/>;
    /// <c>"v"</c> for <see cref="IniEntryKinds.Value"/>.
    /// </returns>
    /// <exception cref="ArgumentOutOfRangeException">Thrown when <paramref name="kind"/> is not a recognised value.</exception>
    public static string MapToCssToken(this IniEntryKinds kind) => kind switch
    {
        IniEntryKinds.Key   => "k",
        IniEntryKinds.Ok    => "ok",
        IniEntryKinds.Value => "v",
        _                   => throw new ArgumentOutOfRangeException(nameof(kind), kind, null),
    };
}
```

### Task 6: Component — EmailSignupForm

**Files:**

- Create: `Source/Presentation.Web/Components/EmailSignupForm.razor`

- [ ] **Step 1: Create `EmailSignupForm.razor`**

```razor
<form class="@FormCssClass" @onsubmit="HandleSubmit" @onsubmit:preventDefault novalidate>
    <div class="signup-row">
        @if (!Compact)
        {
            <input class="signup-name"
                   type="text"
                   placeholder="Your name (optional)"
                   value="@_name"
                   @oninput="@(eventArgs => _name = eventArgs.Value?.ToString() ?? string.Empty)"
                   aria-label="Your name" />
        }

        <input type="email"
               placeholder="you@email.com"
               value="@_email"
               @oninput="HandleEmailInput"
               aria-label="Email address" />

        <button type="submit" class="btn btn-accent">@CallToAction</button>
    </div>

    <div class="signup-err">Hmm, that doesn't look like an email address.</div>

    <div class="micro micro-default">
        <img class="px" src="@Images.Icons.Win95.Information32Gif" alt="" />
        <span>No spam — one note when your beta key is ready. Unsubscribe anytime.</span>
    </div>

    <div class="signup-success">
        <img class="px" src="@Images.Icons.Win95.Mail24" alt="" />
        <div>
            <b>You're on the list@(string.IsNullOrWhiteSpace(_name) ? string.Empty : $", {FirstName}")!</b>
            <span>We'll email <strong>@_email</strong> the moment closed beta keys go out.</span>
        </div>
    </div>
</form>

@code {
    private enum SubmissionStates { Idle, ValidationError, Submitted }

    [Parameter] public string CallToAction { get; set; } = "Request beta access";
    [Parameter] public bool Compact { get; set; }

    private string _name  = string.Empty;
    private string _email = string.Empty;
    private SubmissionStates _submissionState = SubmissionStates.Idle;

    private string FormCssClass => _submissionState switch
    {
        SubmissionStates.ValidationError => "signup err",
        SubmissionStates.Submitted       => "signup done",
        _                                => "signup",
    };

    private string FirstName =>
        _name.Trim() is { Length: > 0 } trimmedName
            ? trimmedName.Split(' ')[0]
            : string.Empty;

    private void HandleEmailInput(ChangeEventArgs eventArgs)
    {
        _email = eventArgs.Value?.ToString() ?? string.Empty;

        if (_submissionState == SubmissionStates.ValidationError)
        {
            _submissionState = SubmissionStates.Idle;
        }
    }

    private void HandleSubmit()
    {
        bool isValidEmail = System.Text.RegularExpressions.Regex.IsMatch(
            _email.Trim(),
            @"^[^\s@]+@[^\s@]+\.[^\s@]+$");

        if (!isValidEmail)
        {
            _submissionState = SubmissionStates.ValidationError;

            return;
        }

        _submissionState = SubmissionStates.Submitted;
    }
}
```
